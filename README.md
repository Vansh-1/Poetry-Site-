# Poetry & Shayari Site (Hindi & English)

This project is a personal website for publishing my poetry, shayari, and short stories in **Hindi** and **English**.

## Goals
- Bilingual content (Hindi and English).
- Author area where I can log in to upload and update my poems, shayari, and stories.
- Public area where visitors can read everything without needing to log in.
- Use a database to store all content securely.

## Next Steps
- Decide on the tech stack for the frontend and backend.
- Design the database schema for poems, stories, users, and tags.
- Implement authentication for the author dashboard.
- Build the public reading experience with browsing and filtering.
